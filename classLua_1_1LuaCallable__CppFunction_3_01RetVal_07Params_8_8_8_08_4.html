Detailed Description

template<typename RetVal, typename... Params>
class Lua::LuaCallable_CppFunction< RetVal(Params...)>

Wraps a std::function, converting arguments and return value to/from the Lua state.

Templated on the parameters and return type of the function. If either the parameters or the return type are not convertable to Lua types, it will fail to compile.

Calls the wrapped function.

Uses the indices trick to number each parameter. From there, reads each argument from the stack and calls the function. Pushes the return value onto the stack.

Exceptions
LuaCppCallErrorThe number of arguments passed from Lua is incorrect.
LuaInvalidStackContentsThe return valud could not be converted to the requested type.
